Connect Telegram group

Setup (use a group where you can add members)

  1. Create a new group in Telegram (you'll be admin). Or use any group where you can add the bot.
  2. Add your bot to the group (Add members → search your bot, e.g. MyVideoSync2025Bot).
  3. Get Chat ID: Add @userinfobot to the same group. It will send a message like Id: -1001275085057. Copy that number (with the minus).
  4. Paste your Bot Token (from @BotFather) below and click Save. Your group Chat ID is already set in the app.
  5. Upload a video in the group (as video message or video file), then click Sync from Telegram. The video will appear on your website.
  6. Auto-forward: When a client uploads a video on P hub, it is automatically sent to your connected Telegram group.

Tip: Use the same URL for admin and viewing (e.g. yoursite.vercel.app). If you sync on localhost, mobile won't see those videos. Prefer Telegram only mode when deploying to Vercel—videos stream from Telegram and work on all devices.
